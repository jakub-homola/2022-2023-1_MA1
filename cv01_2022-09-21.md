### vztahy mezi logickými spojkami

Mějme výroky A, B a C. Pak následující výroky jsou platné:

příklad: přečti to

|||
|:-:|:-|
| $\neg(\neg A) = A$ | dvojitá negace |
| $A \implies B \iff \neg A \vee B$ |  |
| $A \implies B \iff \neg B \implies \neg A$ | věta obměněná |
| $(A \iff B) \iff (A \implies B) \wedge (B \implies A)$ | ekvivalence je obousměrná implikace |
| $\neg(A \wedge B) \iff \neg A \vee \neg B$ | de morganovy zákony |
| $\neg(A \vee B) \iff \neg A \wedge \neg B$ | de morganovy zákony |
| $A \wedge (B \vee C) \iff (A \wedge B) \vee (A \wedge C)$ | distributivní zákony |
| $A \vee (B \wedge C) \iff (A \vee B) \wedge (A \vee C)$ | distributivní zákony |



tautologie
- taková výroková forma, která je vždy pravdivá
- např. $A \vee \neg A$

kontradikce
- taková výroková forma, která je vždy nepravdivá
- např. $A \wedge \neg A$

Kvantifikátory
--------------

přídání před výrokovou formu. tím se specifikují ty volné proměnné a z výrokové formy se stává výrok.

| název | značení | čtení |
|-|:-:|-|
| obecný kvantifikátor | $\forall$ | pro všechna, pro každé |
| existenční kvantifikátor | $\exists$ | existuje |
| kvantifikátor jednoznačné existence | $\exists!$ | existuje právě jeden |

existuje ... takové, že
pro každé ... platí, že

příklady: přečíst, rozhodnout pravdivost
- $\forall n \in \N : x > 5$
    - to řeknu já, napíšu takový ten popis
    - pro každé n z množiny přirozených čísel platí, že n je větší než 5
    - pro každé přirozené číslo n platí, že n je větší než 5
    - každé přirozené číslo je větší než 5
    - nepravda. protipříklad 2
- $\exists n \in \N : x > 5$
    - to zase ještě řeknu já, napíšu takový ten popis
    - existuje n z množiny přirozených čísel takové, že n je větší než 5
    - existuje přirozené číslo n takové, že n je větší než 5
    - některé přirozené číslo je větší než 5
    - pravda, třeba 16
- $\exists! n \in \N : x > 5$
    - existuje právě jedno přirozené číslo n takové, že n je větší než 5
    - existuje právě jedno přirozené číslo větší než 5
    - nepravda. protipříklad 7 a 8
- $\forall x \in \R : x^2 > 0$
    - pro každé reálné číslo x platí, že x na druhou je větší než 0
    - nepravda. protipříklad 0
- $\exists x \in \R : ln(x) = 0$
    - existuje reálně číslo x takové, že přirozený logaritmus x je rovno 0
    - pravda. x = 1
- $\exists! x \in \R : e^x = 1$
    - existuje právě jedno reálné číslo x takové, že e na x je rovno 1
    - pravda. x = 0
- $\forall x \in \R : x > 0 \implies x^3 \ge 0$
    - pro každé reálné číslo x platí, že pokud je x větší než 0, pak x na třetí je větší nebo rovno nule
    - pravda
- $\exists x \in \R : \sin(x) = \frac{\sqrt{2}}{2}$
    - existuje reálné číslo x takové, že sinus x je roven odmocnině ze dvou dělěno dvěma
    - pravda. třeba $\frac{\pi}{4}$ nebo $\frac{3\pi}{4}$
- $\exists! x \in \R : cos(x) = \frac{1}{2}$
    - existuje právě jedno reálné číslo x takové, že sinus x je roven jedné polovině
    - nepravda. nakreslit cosinus. třeba $\frac{\pi}{3}$ nebo $-\frac{\pi}{3}$
- $\exists x \in \R : x^2 - 3x + 2 = 0$
    - existuje reálné číslo x takové, že x na druhou plus tři x mínus dva je rovno nule
    - pravda. 1 a 2
- $\forall x \in \R : -x^2 - x + 6 > 0$
    - pro každé reálné číslo x platí, že mínus x na druhou mínus x plus 6 je větší než 0
    - nepravda. nakreslit graf $-x^2$ a zjistit, že nemůžou být všechny hodnoty nad 0



příklady naopak: říct výrok a ať to napíšou v kvantifikátorech. rozhodni pravdivost.
- pro všechna přirozená čísla $n$ platí, že $n^2 > n$
    - $\forall n \in \N : n^2 > n$
    - nepravda. protipříklad 1
- existuje reálné číslo $x$ takové, že $x^2+x-12=0$
    - $\exists x \in \R : x^2+x-12=0$
    - pravda, 3 a -4
- absolutní hodnota všech reálných čísel je kladná
    - $\forall x \in \R : |x| > 0$
    - nepravda. protipříklad 0
- nějaké přirozené číslo je záporné
    - $\exists n \in \N : n < 0$
    - nepravda
- všechna záporná reálná čísla mají zápornou třetí mocninu
    - $\forall x \in \R : (x < 0) => (x^3 < 0)$
    - pravda
- druhá mocnina všech přirozených čísel větších než 10 je větší než 100
    - $\forall n \in \N : (n > 10) => (n^2 > 100)$
    - pravda



### Negace výroků s kvantifikátorama

- vymění se kvantifikátor (obecný $\leftrightarrow$ existenční) a zneguje se vnitřek
- u kvantifikátoru jednoznačné existence to není tak jechoduché. to si ukážeme později

příklady co udělám já:
- $\forall x \in \R : x > 5$
    - $\exists x \in \R : x \le 5$
- $\exists x \in \R : x \ge 7$
    - $\forall x \in \R : x < 7$

real-world příklady, zneguj:
- všechna jablka jsou sladká
    - existuje jablko, které není sladké
    - nějaké jablko není sladké
- nějaký pomeranč je kyselý
    - všechny pomeranče nejsou kyselé
    - žádný pomeranč není kyselý

další příklady:
- $\forall x \in \R : \sin(x) < 1$
    - $\exists x \in \R : \sin(x) \ge 1$
- $\forall x \in \R : \sqrt{x} \ge 0$
    - $\exists x \in \R : \sqrt{x} < 0 \wedge x \in D\sqrt{}$
- $\exists n \in \N : n^2 = 256$
    - $\forall n \in \N : n^2 \neq 256$



### Více kvantifikátorů

- když je víc stejných kvantifikátorů za sebou sebe tak se mezi němi říká a
- pokud je po obecném kvantifikátoru existenční, neříká se platíže
- závorky se můžou ale nemusí dávat

příklady, přečti a rozhodni pravdivost:
- $\exists x \in \N \enspace \exists y \in \N : x \le y$
    - existuje přirozené číslo x a existuje přirozené číslo y takové, že x je menší nebo rovno než y
    - pravda. třeba x=3 y=5
- $\forall x \in \N \enspace \forall y \in \N : x \le y$
    - pro každé přirozené číslo x a pro každé přirozené číslo y platí, že x je menší nebo rovno než y
    - nepravda. protipříklad x=5 y=3
- $\forall x \in \N \enspace \exists y \in \N : x \le y$
    - pro každé přirozené číslo x existuje přirozené číslo y takové, že x je menší nebo rovno než y
    - pravda. třeba y=x
- $\exists x \in \N \enspace \forall y \in \N : x \le y$
    - existuje přirozené číslo x takové, že pro každé přirozené číslo y platí, že x je menší nebo rovno než y
    - nepravda

pravidla prohazování kvantifikátorů
- rozdílné kvantifikátory obecně nelze mezi sebou prohazovat
    - $\forall x \enspace \exists y   \enspace\;\not\!\!\!\!\!\iff   \exists y \enspace \forall x$   , ale $\impliedby$ platí
- stejné kvantifikátory, které jsou za sebou, lze prohazovat, a můžeme je sloučit
    - $\forall x \enspace \forall y   \iff   \forall y \enspace \forall x   \iff   \forall x,y$
- kvantifikátory které nejsou za sebou nelze prohazovat
    - $\forall x \enspace \exists y \enspace \forall z   \enspace\;\not\!\!\!\!\!\iff   \forall z \enspace \exists y \enspace \forall x$

příklad, uvědom si rozdíl:
- (pro každý zámek) (existuje klíč), který do něho pasuje
- (existuje klíč takový, že) (pro každý zámek platí, že) do něj ten klíč pasuje
    - existuje klíč, který pasuje do každého zámku

příklad, přečti a rozhodni pravdivost:
- $\forall x,y \in \R : x < y  \implies  x^2 < y^2$
    - pro všechna reálná čísla $x$ a $y$ platí, že pokud je $x$ menší než $y$, pak je $x^2 < y^2$
    - nepravda. protipříklad $x=-3$, $y=-2$
- $\forall y \in \R \enspace \exists x \in \R : \tg(x) = y$
    - pro každé reálné číslo $y$ existuje reálné číslo x takové, že $\tg(x) = y$
    - pravda. obor hodnot funkce tangens je $\R$
- $\exists y \in \R \enspace \forall x \in \R^+ : \ln(x) \le y$
    - existuje reálné číslo y takové, že pro každé kladné reálné číslo x platí, že $\ln(x) \le y$
    - nepravda. funkce logaritmus není shora omezená
- $\forall a,b,c \in \R : a^2 + b^2 = c^2$
    - pro všechna reálná čísla $a$, $b$ a $c$ platí, že $a^2 + b^2 = c^2$
    - nepravda. vypadá to sice pythagorova věta, ale ta platí jenom když $a,b,c$ jsou stranypravoúhlého trojúhelníku
- $\exists a,b \in \N : a^2 + b^2 = 25$
    - existují přirozená čísla a a b taková, že $a^2 + b^2 = 25$
    - pravda. třeba $a=3$, $b=4$
- $\exists! x \in \R \enspace \forall y \in \R : x + y = y$
    - existuje právě jedno reálné číslo x takové, že pro každé reálné číslo y platí, že $x + y = y$
    - pravda. $x=0$



### Negace výroků s více kvantifikátorama

- prostě to co je "uvnitř" vnějšího kvantifikátoru se bere jako prostě celá vnitřní výroková forma která se pak zase zvlášť rekurzivně neguje

příklad: zneguj předchozí výroky, kromě toho s tangens, ten je těžší (je tam ještě ten definiční obor)
- $\forall x,y \in \R : x < y  \implies  x^2 < y^2$
    - $\exists x,y \in \R : x < y \wedge x^2 \ge y^2$
- $\forall y \in \R \enspace \exists x \in \R : \tg(x) = y$
    - $\exists y \in \R \enspace \forall x \in \R : \tg(x) \neq y \vee x \notin D\tg$
- $\exists y \in \R \enspace \forall x \in \R^+ : \ln(x) \le y$
    - $\forall y \in \R \enspace \exists x \in \R^+ : \ln(x) > y$
- $\forall a,b,c \in \R : a^2 + b^2 = c^2$
    - $\exists a,b,c \in \R : a^2 + b^2 \neq c^2$
- $\exists a,b \in \N : a^2 + b^2 = 25$
    - $\forall a,b, \in \N : a^2 + b^2 \neq 25$
- $\exists! x \in \R \enspace \forall y \in \R : x + y = y$
    - $(\forall x \in \R \enspace \exists y \in \R : x+y=y) \vee (\exists x_1, x_2 \in \R \enspace \exists y \in \R : x_1+y=y \wedge x_2+y=y \wedge x_1 \neq x_2)$
    


### Kvantifikátor jednoznačné existence a jeho negace

kvantifikátor jednoznačné existence se dá přepsat za použití jenom obecného a existenčního. Pomocí toho se dá i negovat. Mějme výrokovou formu V(x)
- $\exists! x : V(x) \iff ((\exists x : V(x)) \wedge (\forall x,y : V(x) \wedge V(y) \implies x = y))$

příklady:
    A = {2, 4, 5} // výčtem prvků
    B = {1, 2, 3, ...}
    C = {x \in R : x > 470} // pomocí charakteristivké vlastnosti
    D = {kolečko, čtvereček, trojúhelníček}

a \in A

diagramy!!!

podmnožina (subset)
    nechť A a B jsou množiny.
    řekneme, že množina A je podmnožinou množiny B
        <=>
    \forall x \in A : x \in B
    a značíme A \subset B
    všechny značení
    spojitost s implikacema
        A \subset B
            <=>
        \forall x : x \in A => x \in B
    příklad:
        A = { x \in R : U(x) }
        B = { x \in R : V(x) }
        odněkud víme, že A \subset B
        y \in R a U(y) je pravda
        co můžeme říct o V(y)?

        A \subset A?
        \emptyset \subset A?

rovnost množin
    nechť A a B jsou množiny.
    řekneme, že množina A je rovna množině B
        <=>
    A \subset B azároveň B \subset A
    a značíme A = B

operace na množinách
    nechť A a B jsou množiny.
    sjednocení množin (union)
        A \cup B = { x : x \in A OR x \in B }
        x \in (A \cup B)   <=>   x \in A OR x \in B
    průnik množin (intersection)
        A \cap B = { x : x \in A AND x \in B }
        x \in (A \cap B)   <=>   x \in A AND x \in B
    rozdíl (set difference)
        A \ B = { x : x \in A AND x \notin B }
        x \in (A \ B)   <=>   x \in A AND x \notin B
    doplněk (complement)
        A' = { x : x \notin A }
        x \in A'   <=>   x \notin A

příklad, množiny A={2,4,5}, B={1,2,3} a všechny operace









logická výstavba matematiky // tohle by kdyžtak bylo lepší do dalšího cvika, jestli vůbec to je třeba
===========================

základy co je definice, co je věta (theorem), důkaz (proof), axiom

axiom
    výrok, jehož pravdivost se předpokládá. je pravdivý "protože jsem to tak řekl", "protože prostě proto", "protože chci"

věta (theorem)
    obvykle ve formě implikace nebo ekvivalence
    pravdivé výroky, které lze pomocí pravidel logiky odvodit z axiomů a již dříve dokázaných vět
    předpoklady a tvrzení
        nechť ... . potom ...
        ... => ...
    dva způvoby psaní předpokladů
        (\forall x \in R) (\forall n \in N) : tvrzení
        nechť x \in R a n \in N. pak tvrzení
    má obvykle smysluplné pojmenování a usnadňuje danou problematiku
        není to jenom pojmenovaná sada písmenek a logických značek

důkaz (proof)
    logické deduktivní odvození výroku z jiných pravdivých výroků
    rozpitvání věty (velké složité implikace u které nemusí být vůbec jasné proč to má platit) pomocí logických kroků na sérii jednodušších implikací (vět), které jsou již dokázány
    typy důkazů:
        přímý: předpoklady => ... => tvrzení
        nepřímý: not(tvrzení) => ... => not(předpoklady)
        sporem: předpoklady AND not(tvrzení) => ... => spor
        indukcí: \forall n \in N : ...

příklad: AG nerovnost
    úvod:
        aritmetický průměr (x+y)/2
        geometrický průměr sqrt(x*y)
    věta:
        \forall x,y \in R, x,y>=0 : (x+y)/2 >= sqrt(xy)
        nebo taky
        nechť x,y \in R, x,y >= 0
            => pak
        (x+y)/2 >= sqrt(xy)
    důkaz:
        vyjdu z něčeho co už vím
        0 <= (x-y)^2 = ... = (x+y)^2 - 4xy
        =>
        4xy <= (x+y)^2
        => // všechno je kladné, sqrt je rostoucí takže neotáčí nerovnost
        2sqrt(xy) <= sqrt((x+y)^2) = (x+y)
        =>
        (x+y)/2 >= sqrt(xy)

definice
    zavedení pojmenování pro nějakou vlastnost, strukturu, ...
    obvykle ve formě ekvivalence
    příklad:
        řekneme, že celé číslo k je SUDÉ
            <=>
        \exists l \in Z : k = 2l

takhle se buduje matematika. začíná se z axiomů (standartně teorie množin (ZFC set theory)), na které navazují složitější věty
definovat přirozená čísla a dokázat, že 1+1=2 dá hodně práce! (peano axioms)
