Tečna ke grafu funkce
---------------------

Rovnici tečny ke grafu funkce $f$ v bodě $x_0$ lze vyjádřit jako

$t(x) = f(x_0) + f^\prime(x_0)(x-x_0)$

### Příklady: najděte rovnici tečny ke grafu funkce $f$ v bodě $x_0$
- $f(x) = x^2 + 3, \quad x_0=1$
    - $t(x) = 2x + 2$
- $f(x) = sin(x), \quad x_0=\frac{\pi}{6}$
    - $t(x) = \frac{\sqrt{3}}{2}x + \frac{1}{2} - \frac{\sqrt{3}\pi}{12}$



l'Hospitalovo pravidlo
----------------------

Při výpočtu limity funkce, kdy po dosazení vychází $\frac{0}{0}$ nebo $\frac{cokoliv}{\infty}$, můžeme použít l'Hospitalovo pravidlo:

> $"\lim_{x \rightarrow x_0} \frac{f(x)}{g(x)} = \frac{0}{0} \quad \text{nebo} \quad \frac{cokoliv}{\infty} \quad "$  
> $\quad \implies$  
> $\lim_{x \rightarrow x_0} \frac{f(x)}{g(x)} = \lim_{x \rightarrow x_0} \frac{f^\prime(x)}{g^\prime(x)}$, pokud limita na pravé straně existuje

### Příklady: vypočtěte limitu funkce pomocí l'Hospitalova pravidla
- $\lim_{x \rightarrow 0} \frac{\sin(x)}{x}$
    - $1$
- $\lim_{x \rightarrow 0} \frac{1-\cos(x)}{x^2}$
    - $\frac{1}{2}$
- $\lim_{x \rightarrow 0} \frac{e^x-1}{x}$
    - $1$
- $\lim_{x \rightarrow 0} \frac{e^{2x}-2x-1}{\sin^2(3x)}$
    - $\frac{2}{9}$
- $\lim_{x \rightarrow 0+} x \ln(x)$
    - $0$



Intervaly ryzí monotonie
------------------------

věta: nechť funkce f je spojitá v intervalu I a krajními body a,b \in R^*
\forall x \in (a,b) : f'(x)>0
    =>
funkce f je na intervalu I rostoucí
podobně klesající, neklesající, nerostoucí, konstantní

souhrnný pojem pro nerostoucí a neklesající - monotónní
souhrnný pojem pro rostoucí a klesající - ryze monotónní
note: ryze monotónní => monotónní

note: jenom jednosměrná implikace. příklad x^3, která má nulovou derivaci v 0, ale je rostoucí na celém R
    na sebe navazující intervaly se dají někdy spojovat ()

příklad
najdi intervaly ryzí monotonie funkce f(x) = 2x^3 - 3x^2 - 12x + 1
hledáme intervaly RYZÍ monotonie, takže hledáme intervaly, kde je derivace > nebo < 0

Lokální extrémy
---------------

definice:
    f má v x_0 lokální maximum <=> \exists epsilon>0 tž. \forall x \in R : 0<|x-x_0|<epsilon  =>  f(x) <= f(x_0)
    lokální minimum
    ostré lokální maximum
    ostré lokální minimum
obrázek

věta: nutná podmínka existence lokálního extrému
    f má v x_0 lokální extrém
        =>
    f'(x_0) = 0 nebo f'(x_0) neexistuje
tedy lokální extrémy jsou jenom tam, kde je derivace nulová, nebo tam kde derivace neexistuje. nikde jinde.

věta
pokud je funkce diferencovatelná, pak má lokální extrémy jenom tam, kde je derivace nulová (ve stacionárních bodech)
    diferencovatelná <=> má všude derivaci, která je konečná

body, ve kterých je derivace nulová = stacionární body
    x_0 je stacionárním bodem funcke f <=> f'(x_0) = 0
    // kandidáti na lokální extrémy

věta: postačující podmínka existence lokálního extrému
    f'(x_0) = 0   &&   f''(x_0) > 0
        =>
    f má v x_0 ostré lokální minimum
    apod. pro maximum
no jo ale co třeba f(x)=x^6 ?
dále:
    pokud jsou všechny derivace až po (n-1)-ní nulové a nenulová je až n-tá derivace, pak
        v x_0 JE lokální extrém <=> n je sudé
            f^(n)(x_0) > 0   =>   minimum
            f^(n)(x_0) < 0   =>   maximum
        pokud je n liché, pak v x_0 NENÍ extrém

příklad: najdi všechny lokální extrémy funkce f(x) = x^3 + 3x^2 - 1
derivace
ta má všude smysl, a je konečná => funkce je diferencovatelná, takže lokální extrémy jsou jenom ve stacionárních bodech
derivace rovno nule, vyřešit
druhá derivace, hodnoty ve stacionárních bodech, podle toho min/max
